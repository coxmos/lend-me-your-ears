「耳朵借我」找過不少音樂人談他們的音樂人生，我總請他們開個12到15首的歌單，先傳給我。旭章是目前為止唯一不但傳了歌單，還為這些歌加上註釋的來賓，好貼心。
認識旭章是<a href="https://www.youtube.com/watch?v=cjNKRy8791k">2012年我在「音樂五四三」訪問張懸</a>，他帶著樂器來現場替她伴奏，那幾年他是Algae的團員。這次訪問我才知道，旭章回台灣決心投入音樂工作，第一位合作的歌手就是張懸 / 安溥，恐怕也是影響他最大的歌者了。今年大為轟動的安溥「煉雲」演唱會，旭章也是樂手之一，表現之精采有目共睹。接下來他們還有一系列演出要同台，我很期待。
後來這幾年，我們偶爾會在公部門評審的場合遇見，聽他說最近做的事，總在各種跨界實驗中遊走。聽他分享對樂壇生態的觀察，也頗有見地。今年七月，他和流氓阿德一起來上節目，也是擔任伴奏。我們又聊了聊，欲罷不能，便約了專心來上一集節目。
當然，旭章的身分遠不只樂手。他能唱，能作詞作曲，也會編曲製作。雖然現在他都在彈鍵盤，但當年他可是大學吉他社的社長，手底功夫不含糊的。旭章在香港出生，從小住過香港、上海、台灣、美國，家境是很不錯的，長輩從小讓他學琴，要把他培養成菁英中的菁英，期許他以後要在每個世界大國的首都都有頂尖的工作經驗（旭章笑著說：這個願望後來真的實現了，只不過是當樂手）。一路讀書，出國留學，旭章終於確定他不想走爸媽期待他走的那條白領菁英之路，決心要做音樂，於是鬧了家庭革命，兩代人的裂痕一直到最近才慢慢修補。
既然下定決心，就得為自己的決定負責。旭章這些年做了很多工作，組了「守夜者」樂團，為遊戲製作配樂和主題曲，為電影、電視、網劇寫配樂插曲主題曲，擔任歌手的配唱製作、編曲、演唱會樂手。大抵這一行的每個環節，他都做過了。面對嚴酷的環境，旭章和他的同輩音樂人並沒有時間和心情慨嘆。他們互相串聯成能夠互相隨時支援的團隊，工作起來既有才情也有效率，我在好幾個領域：設計、影像、音樂圈，都看到了這樣的現象，還滿感動的。旭章說，他始終是焦慮的，不斷在想還可以做什麼，以及自己真正最應該做什麼。我感覺他在唱片工業崩塌之後的環境裡，一面不斷接活來幹，一面在重新組織自己的能力，正在摸索最好的突破口。
底下分享旭章的歌單，以及他的註解。這集節目訊息量超大，請張開耳朵專心聽。
from 旭章：
這分歌單是旭章參與電影、遊戲等跨界創作的不同主題曲創作，以及為合作歌手們彈琴填詞的各種記錄，在這些音樂中我們好像失去了時間 ，也忘了這時代各種音樂平台的改變。 不變的是，自己剛剛好載浮載沉的位置，永遠也必須真心依靠音樂給予出口。
<strong>1. Paul McCartney / Vanilla Sky :
</strong>電影香草天空的主題曲,第一首啟蒙也鼓勵自己有一天要為電影創作歌曲的心。
<strong>2. Mondialito (夢的亞朵) / Rainy green grass
</strong>他們是影響自己的重要「組合」 ，這兩個字也讓守夜人樂團有著各種跨界的影響啟蒙。
<strong>3. 黃妃 / 唱乎爽 
</strong>為電影《五星級魚干女》創作，此電影音樂在2017年入圍了電視金鐘獎
<strong>4. 張懸 / 如何
</strong>第一首跟安溥完成的錄音作品。
<strong>5. 守夜人 / 守夜慶典
</strong>第一首在廠牌發行下的演唱歌曲 ，為雷亞遊戲的世界觀打造的創作。
<strong>6. 林宥嘉 / 船
</strong>嘗試將守夜人的「助眠」、「守護」的概念參與製作與編曲 （與陳君豪林宥嘉共同製作） 。也是他9週年的一首特別紀念歌曲。
<strong>7. 關淑怡 / 可惜 
</strong>此歌曲是在擔任大學吉他社社長時 最常哼唱的歌，也開啟自己接受華語音樂原來有很多不一樣但有得選擇的一個重要的歌。
<strong>8. 陳如山 / 黑白</strong>
當初為電影《黑白》所創作的主題曲，嘗試為創作歌手四分衛陳如山創作， 與電影劇情激盪初新的想法。是第一首為電影量身打造的創作。
<strong>9. 小球 / 替
</strong>此時正在上映的電影《粽邪》主題曲， 呈現了恐怖情節之外的另一種遺憾與期盼。無論是電影中或現實生活，每個人在生活裡總有完成不了的事情，都變成遺憾，希望有一首歌可以安慰這樣的遺憾。
<strong>10. 這群人 展榮展瑞 / 閃退
</strong>在2018年自己的作詞作品，也是在網紅文化興起時與他們的跨界創作。雖然有點跳tone，卻是另一個世界視野。
<strong>11. 22 cats / 電影人生
</strong>20多歲時與香港的樂團一起玩樂團的時光，如今他們在自立自強的維港唱片品牌下發行一首一首純粹indie的作品。
<strong>12. 朴樹 / 今夜的滋味</strong>
<strong>13. 守夜人 ft. 柯泯薰 / 倒數開始
</strong>一首在各種節慶中寫給自己的歌 （也是第一次為自己而寫的歌）。
你知道嗎？閉著眼睛從10數到1，以自己的速度，不論快慢，當你專注於數數的時候，某些願望就已慢慢成形，當你專注數著，本想著某些煩雜的事，會隨著數的數字一個一個消失不見…
我們終於可以 孤僻而快樂的落單 。
